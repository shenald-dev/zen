# Zen

> Find your flow. A minimalist deep-work timer for focused building.

Zen helps you implement the Pomodoro Technique with a clean, distraction-free interface. No notifications, no clutter — just you and your work.

![Python](https://img.shields.io/badge/Python-3.8%2B-blue?logo=python)
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

## Features

- Pomodoro timer (25min work, 5min break)
- Session tracking and analytics
- Desktop notifications (optional)
- Configurable intervals
- Terminal-based, no GUI needed

## Installation

```bash
pip install zen-timer
```

## Usage

```bash
zen start
```

Then focus. Zen will ring when it's time for a break.

## Why I Built Zen

I kept getting distracted by complex productivity apps. I needed something that *stays out of my way* and just tells me when to work and when to rest. So I built it.
