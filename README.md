# Zen

🧘 **Find your flow.**

Zen is a minimalist CLI tool designed to help you enter and maintain deep work states. It provides a focused environment for high-performance building.

## 🚀 Quick Start

```bash
python src/main.py [minutes]
```

## 🛠️ Features

- **Flow Timer:** Visual progress tracking for your deep work sessions.
- **Minimalist Aesthetic:** Clean, distraction-free terminal UI.
- **Intentional Design:** Built to facilitate the "Vibe Coding" philosophy.
